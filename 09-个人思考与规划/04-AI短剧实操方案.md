## 第二步:写剧本(30分钟)

### ChatGPT Prompt模板
```
请帮我创作一个短剧剧本:
- 类型: [霸道总裁/甜宠/悬疑/复仇/逆袭]
- 时长: 60秒
- 场景数: 3-4个
- 人物: 2-3个主要角色
- 核心冲突: [一句话说明矛盾点]
- 反转点: 在第[X]秒出现

要求:
1. 每个场景标注镜头(特写/中景/远景)
2. 标注人物表情和动作
3. 对白要简短有力
4. 结尾留悬念/金句
```

### 输出格式示例
```
【场景1 - 豪华办公室 - 远景 - 5秒】
霸道总裁坐在办公桌前,冷眼看着文件
总裁(冷漠):"她的资料删掉,我不想再看到。"

【场景2 - 电梯口 - 中景 - 8秒】
女主匆忙跑进电梯,与总裁四目相对
女主(紧张):"对...对不起!"
(电梯门缓缓关闭,总裁若有所思)

【场景3 - 咖啡厅 - 特写 - 10秒】
...
```

## 第三步:生成角色图(1小时)

### Midjourney角色生成流程

**1. 设定角色形象**
```
/imagine prompt:
A handsome Chinese CEO, 30 years old, wearing black suit,
cold expression, sharp eyes, office background,
professional photography, ultra realistic, 8k --ar 9:16 --v 6
```

**关键参数:**
- `--ar 9:16`: 竖屏比例(抖音/快手格式)
- `--v 6`: 最新版本模型
- `--s 750`: 风格化程度(可选)

**2. 保持角色一致性**
- 生成满意角色后,点击图片右键保存
- 在Discord中上传该图片
- 复制图片链接
- 使用 `图片链接 + 新场景描述` 保持同一角色

示例:
```
/imagine prompt:
https://cdn.midjourney.com/xxx.png
同一个男人,在咖啡厅,温柔微笑,休闲装,
warm lighting, cinematic --ar 9:16 --v 6
```

**3. 生成场景背景**
```
/imagine prompt:
luxury office interior, floor-to-ceiling windows,
city view, modern furniture, golden hour lighting,
architectural photography, 8k --ar 9:16 --v 6
```

## 第四步:图片转视频(2小时)

### 可灵AI操作流程

**1. 上传图片**
- 登录可灵AI
- 点击"图生视频"
- 上传Midjourney生成的角色图

**2. 写运镜提示词**
```
镜头从远景推进到中景,
人物眼神从冷漠转为惊讶,
背景虚化,光线逐渐温暖
```

**运镜类型:**
- 推镜:由远到近
- 拉镜:由近到远
- 摇镜:左右或上下移动
- 跟镜:跟随人物移动
- 升降:镜头上升或下降

**3. 生成设置**
- 时长:5秒(推荐)
- 镜头运动:中等
- 点击生成(每次消耗1次免费额度)

**4. 批量生成**
每个场景生成2-3个版本,挑选最好的

## 第五步:配音(30分钟)

### 方案A:剪映AI配音(推荐)
1. 打开剪映 → 开始创作
2. 导入视频素材
3. 点击"文本" → "智能字幕" → "自动识别"
4. 手动输入对白文本
5. 点击"文本朗读"
6. 选择声音:
   - 男声:磁性男声/成熟男声
   - 女声:甜美女声/温柔女声
7. 调整语速:1.0-1.2倍
8. 调整音量:建议80%

### 方案B:真人配音
- 使用手机录音
- 或在淘宝找配音员(5-20元/条)

## 第六步:剪辑成片(2小时)

### 剪映剪辑流程

**1. 导入素材**
- 将所有视频片段按顺序导入时间轴

**2. 精细剪辑**
- 卡点:对白与画面完美对应
- 转场:使用"淡入淡出"(不要用太花哨的)
- 节奏:每个镜头3-5秒为宜

**3. 添加字幕**
```
字幕样式推荐:
- 字体:思源黑体/阿里巴巴普惠体
- 大小:居中,占屏幕1/5高度
- 颜色:白色字体+黑色描边
- 特效:逐字出现
```

**4. 背景音乐**
- 剪映音乐库搜索关键词:
  - 霸总剧:大气、磁性
  - 甜宠剧:温馨、浪漫
  - 悬疑剧:紧张、诡异
- 音量调至30%(不要盖过人声)

**5. 调色**
- 使用滤镜:"电影感" 或 "复古"
- 亮度:+10
- 对比度:+15
- 饱和度:+5

**6. 片头片尾**
- 片头(3秒):标题+吸引语
- 片尾(2秒):引导关注/点赞

## 第七步:发布(30分钟)

### 抖音发布技巧

**1. 视频要求**
- 分辨率:1080x1920
- 帧率:30fps
- 时长:60-90秒(第一集可短至45秒)

**2. 标题公式**
```
[反转点] + [情绪词] + [悬念]

示例:
"霸总在电梯偶遇灰姑娘,下一秒却..."
"她以为自己是替身,没想到真相竟是..."
"离婚当天,前夫跪着求复合"
```

**3. 话题标签**
```
#短剧 #霸道总裁 #甜宠 #[你的剧情类型]
```

**4. 发布时间**
- 工作日:12:00-13:00(午休) / 18:00-20:00(下班)
- 周末:10:00-11:00 / 15:00-17:00 / 19:00-22:00

**5. 连载策略**
- 第1集:60秒完整小故事(可独立观看)
- 第2-10集:每集结尾留悬念
- 每天发1-2集,保持用户期待

## 第八步:数据优化(持续)

### 关键数据指标
1. **完播率 >60%**: 前3秒必须抓人
2. **点赞率 >5%**: 情绪点要足够强
3. **评论率 >2%**: 结尾留争议话题
4. **转发率 >1%**: 共鸣或实用价值

### 优化策略
**如果完播率低(<40%):**
- 前3秒改为最高潮片段
- 删减拖沓镜头
- 加快节奏(每镜头<5秒)

**如果互动率低:**
- 结尾添加互动问题
- 评论区引导讨论
- 设置争议性剧情

## 常见问题

**Q1: 可灵AI生成视频总失败怎么办?**
- 提示词要简单明确,不要太复杂
- 人物动作幅度不要太大
- 避免多人物同框
- 分时段使用(避开高峰期)

**Q2: Midjourney生成的角色每次都不一样?**
- 使用图片引用(图片链接+提示词)
- 或使用角色一致性功能 `--cref`
- 或直接使用真人照片(注意版权)

**Q3: 视频没播放量怎么办?**
- 前3秒必须是高潮/冲突点
- 标题要有悬念
- 发布后第一时间回复评论
- 使用热门话题标签

**Q4: 一天能做几集?**
- 熟练后:2-3小时/集
- 新手:4-6小时/集
- 批量制作效率更高

**Q5: 需要什么配置的电脑?**
- 最低:8G内存+独立显卡
- 推荐:16G内存+GTX1650以上
- Mac用户:M1芯片以上

## 成功案例拆解

### 案例1:霸总短剧
- **数据**: 单集播放300万+,涨粉5万
- **成功要素**:
  - 前3秒就是霸总壁咚镜头
  - 对白简短有力("我的女人,谁敢碰?")
  - 每集必有反转

### 案例2:复仇逆袭
- **数据**: 10集总播放2000万+
- **成功要素**:
  - 女主从卑微到强势的转变
  - 每集结尾留悬念
  - 人设讨喜(观众想为她出气)

### 关键启发
1. **前3秒决定生死**: 必须是最冲突的画面
2. **人设要极致**: 霸总就要霸到底,小白花就要惨到底
3. **节奏要快**: 每5秒一个小转折
4. **结尾要狠**: 让观众忍不住看下一集

## 进阶技巧(第二个月)

### 1. 多账号矩阵
- 同一短剧发到3-5个账号

- 分析同类爆款视频
- 拆解其剧情结构

### 3. IP化运营
- 固定角色形象
- 系列化剧情
- 建立粉丝社群

### 4. 工作流优化
- 建立素材库(音效/特效/模板)
- 使用批量渲染
