## 一、剧本创作提示词

### 1.1 霸道总裁剧本

```
你是一位专业的短剧编剧,擅长创作霸道总裁类型的短剧。

请为我创作一个60秒的短剧剧本,要求:

【剧本类型】霸道总裁 + 甜宠
【核心冲突】女主被误认为是新来的清洁工,霸总对她一见钟情
【人物设定】
- 男主:冷酷霸道总裁,实则内心柔软
- 女主:名牌大学毕业生,因为穿着朴素被误会

【剧本要求】
1. 时长:60秒,分为4个场景
2. 每个场景标注:地点、时间、镜头类型(特写/中景/远景)
3. 对白要简短有力,每句话不超过15个字
4. 必须有一个反转点(在第40秒左右)
5. 结尾留悬念,引导观看下一集

【输出格式】
场景1 [地点] [镜头类型] [时长]
环境描述:
人物动作:
对白:

请开始创作。
```

### 1.2 复仇逆袭剧本

```
你是一位擅长创作爽文短剧的编剧。

请为我创作一个"女主复仇逆袭"的短剧剧本。

【背景设定】
女主曾被豪门婆婆羞辱,离婚后发现自己是失散多年的千金大小姐,三年后携巨款回归复仇。

【剧本要求】
1. 时长:90秒
2. 场景数:5个
3. 核心冲突:前婆婆来求女主帮忙,女主冷笑拒绝
4. 情绪爆点:女主说出金句"你当初怎么对我的,记得吗?"
5. 反转:前夫跪地求原谅,女主已经有了新的霸总未婚夫

【人物性格】
- 女主:从隐忍到强势,有仇必报
- 前婆婆:势利眼,现在落魄求人
- 前夫:后悔,但已经晚了
- 新男友:霸总,护妻狂魔

【镜头要求】
- 女主霸气时刻用特写
- 前婆婆跪地用俯视镜头
- 新男友出场用慢镜头

输出格式:
【场景X】地点 | 时长 | 镜头
画面:
动作:
台词:
情绪:

开始创作。
```

### 1.3 甜宠校园剧本

```
创作一个校园甜宠短剧剧本。

【剧情梗概】
学霸男主暗恋学渣女主三年,终于在毕业前鼓起勇气表白。

【场景设定】
1. 教室(放学后)
2. 操场(夕阳下)
3. 校门口(告别时刻)

【情感基调】
青涩、温柔、甜蜜,有笑有泪

【剧本要求】

- 男主性格:外冷内热,不善言辞但行动派
- 女主性格:开朗大方,粗线条但心思细腻

【镜头语言】

请按照以下格式输出:
【场景1】地点 + 氛围 + 时长
画面:
人物:
台词:
情绪:
```

### 1.4 悬疑推理剧本

```
你是一位擅长悬疑推理的编剧。

创作一个60秒的悬疑短剧,要求必须有反转。

【剧情核心】
女主在出租屋发现一只被锁住的箱子,打开后里面是一张自己的照片。

【悬疑设定】
1. 前30秒:营造恐怖氛围,让观众以为是凶杀案
2. 后30秒:反转,原来是男友准备的求婚惊喜

【氛围要求】
- 前半段:压抑、紧张、恐怖
- 后半段:温馨、浪漫、感动

【场景】
- 昏暗的出租屋
- 箱子特写
- 照片特写
- 男友出现

【镜头技巧】
- 手持镜头营造不安感
- 特写放大细节
- 快速剪辑增加紧张感
- 反转时用柔光

【台词】
- 前半段独白(女主内心旁白)
- 后半段对白(男友表白)

输出格式:
【时间轴】0-15秒
场景:
画面:
音效:
台词/旁白:
情绪:

开始创作。
```

### 1.5 通用剧本生成模板

```
我要创作一个短剧,请帮我生成剧本。

【基本信息】
类型: [霸总/复仇/甜宠/悬疑/搞笑]
时长: [60秒/90秒/120秒]
目标平台: [抖音/快手/视频号]

【剧情要素】
主题: [一句话说明核心矛盾]
人物: [2-3个主要角色]
冲突: [主要矛盾是什么]
反转: [在哪里反转,反转是什么]
结局: [开放式/圆满/悬念]

【创作要求】
1. 分镜明确(3-5个场景)
2. 每个场景标注时长
3. 对白口语化,每句不超过20字
4. 高潮在第40-50秒
5. 结尾必须有钩子(引导看下一集)

【镜头要求】
- 特写用于表情和情绪
- 中景用于对话和互动
- 远景用于环境和氛围

【情绪曲线】
开场(吸引) → 冲突(升级) → 高潮(爆发) → 反转(惊喜) → 结尾(悬念)

请按照标准编剧格式输出剧本。
```

## 十、常见问题与解决方案

### Q1: Midjourney角色不一致怎么办?

**解决方案:**
1. 使用 `--cref` 参数引用第一张图
2. 保存并复用详细的角色描述
3. 使用 `--sref` 参数保持风格一致
4. 生成后用PS/剪映统一调色

---

### Q2: 可灵AI生成的视频人物动作不自然?

**解决方案:**
1. 降低动作幅度描述(避免大幅度动作)
2. 使用"缓慢"、"轻微"等限定词
3. 分段生成(一个镜头只做一个动作)
4. 后期剪辑拼接

---

### Q3: 剧本太平淡,怎么增加冲突?

**ChatGPT优化Prompt:**
```
我的剧本如下:
[粘贴剧本]

问题:情节太平淡,缺少冲突

请帮我:
1. 找出3个可以加强冲突的点
2. 增加一个反转

重新输出优化后的剧本。
```

## 十一、提示词使用建议

### 1. 迭代优化
- 第一次生成不满意,要明确指出问题
- 让AI针对性优化,而不是全部重新生成

### 2. 细节描述
- 越详细越好,包括表情、动作、情绪
- 使用形容词增强表现力

### 3. 参考示例
- 给AI提供参考案例("类似XX的风格")
- 可以截图或描述你喜欢的画面

### 4. 分步执行
- 不要一次性要求太多
- 先剧本→再角色→再场景→再分镜

### 5. 保存模板
- 把好用的提示词保存下来
- 建立自己的提示词库
